# Typing Speed Test ⌨️

This is a simple Typing Speed Test web application.
It helps users practice typing and measure their typing speed and accuracy.

## Features

* Displays a random text for typing
* Calculates typing speed (WPM)
* Shows typing accuracy
* Simple and user-friendly interface
* Instant result after completing the test

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Run **index.html** in any web browser.
4. Start typing and check your typing speed.
